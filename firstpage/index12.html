Псевдо- Елементи

CSS псевдо-елементи се използват за добавяне на специални ефекти към някои селектори.
Синтаксисът на псевдо-елемента:

1selector::pseudo-element {
2property:value;
3}

:: first line псевдо-елемента

:: first line псевдо-елемента се използва за добавяне на специален стил на първата линия на един текст.

1 p::first-line {
2 color: #fff0000;
3 font-variant: small-caps;
4 }

::first-letter псевдо-елемента

::first-letter псевдо-елемента се използва за добавяне на специален стил на първата буква от текста.

1 p::first-letter {
2 color: yellow;
3 font-size: xx-large;
4 }

CSS класове

Примерът по-долу ще покаже първата буква на всеки параграф, който е в статия – в червено.

2 p.article::first-letter {color:#ff0000;}
